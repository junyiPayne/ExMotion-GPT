# 科研版测试用例 (Research Test Cases)

**测试重点**: 特征工程准确性、混合模型维度匹配、实验脚本逻辑。

## 1. 特征工程测试

| ID | 测试项 | 步骤 | 预期结果 |
| :--- | :--- | :--- | :--- |
| **RTC-01** | **FFT 频谱计算** | 1. 输入已知频率(如50Hz)的正弦波。<br>2. 调用 `compute_baseline_features`。 | 返回的 MNF (平均频率) 应接近 50Hz，验证算法正确性。 |
| **RTC-02** | **特征维度检查** | 1. 输入 `(Batch, 3, 100)`。<br>2. 计算特征。 | 输出维度应为 `(Batch, 3, 5)` -> Flatten后为 `(Batch, 15)`。 |

## 2. 混合模型测试

| ID | 测试项 | 步骤 | 预期结果 |
| :--- | :--- | :--- | :--- |
| **RTC-03** | **Early Fusion 拼接** | 1. 实例化模型，`feature_dim=15`。<br>2. 传入 raw 和 feat 张量。 | 模型内部在分类层前，Tensor 形状应为 `(Batch, 256+15)`。 |
| **RTC-04** | **消融开关测试** | 1. 设置 `use_lstm=False`。<br>2. 前向传播。 | 模型应跳过 LSTM 层，直接将 CNN 输出 Flatten 后与手工特征拼接。 |

## 3. 实验脚本测试

| ID | 测试项 | 步骤 | 预期结果 |
| :--- | :--- | :--- | :--- |
| **RTC-05** | **自动化流程** | 1. 运行 `experiment_runner.py`。<br>2. 设置极少 Epoch。 | 程序应依次输出 3 组实验结果，并打印最终的 Pandas 对比表格。 |
