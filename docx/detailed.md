# 实验设计与验证协议 (Research Protocol)

## 1. 数据采集流程
本研究严格遵循以下协议以确保数据质量 [cite: 257-268]:
1. **MVIC 标定**: 采集最大自主等长收缩力，用于信号归一化。
2. **防疲劳机制**: 
    * 动作间休息: 5 min
    * 轮次间休息: 30 min
3. **采集动作**: RS, MA, RM, DLB (7.5kg), SLB (25kg)。

## 2. 数据处理管线
* **滤波**: 50Hz 陷波 + 20-95Hz 带通 [cite: 296]。
* **切片**: 窗口 500ms, 重叠 450ms (步长 50ms) [cite: 298]。
* **特征工程**: 
    * 时域: RMS, P-P, RSS
    * 频域: MNF, MDF
    * 倒谱: MFCC (系数 1, 3) [cite: 337]。

## 3. 消融实验设计
| 实验组 | 配置 | 目的 |
| :--- | :--- | :--- |
| **CNN_Only** | 仅卷积层 | 验证局部特征有效性 |
| **BiLSTM_Only** | 仅时序层 | 验证时序依赖有效性 |
| **Proposed** | CNN+BiLSTM+Attn | 验证完整架构性能 |
